### las pruebas que se deberian realizar con la herramienta Postman 
- Get: obtenemos la informacion que nos arroja la api seleccionada por medio de la url http
- Post: realizamos un cambio con la informacion que nos brinda el json y le damos Send para que mande la peticion a la BD
- Put: es prancticamente lo mismo le damos Send para realizar los cambios
- Delete: aqui se aplica un borrado de un registro especifico en el Json aplicamos o especificamos el registro que deseas borrar y revisamos la BD que si realmente el registro esta eliminado
